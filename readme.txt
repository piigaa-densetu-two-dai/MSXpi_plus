～ ピーガー伝説➋代の「MSXπ✨」～
personal use only

「MSXπ✨」は各種カートリッジエミュレーターの「MSXπ typeA」に「✨」を追加したものです。
※「✨」が表記困難な場合は「+」又は「plus」と表記します。

「MSXπ✨」には「MSXπ typeA完全互換モード」と「MSXπ✨モード」2つのモードがあります。
モードは基板上のSOUNDスイッチで設定します。

○スイッチ設定とモードの関係

＊SOUNDスイッチ「0」側 -> MSXπ typeA完全互換モード
既存のMSXπ typeA用のファームウェアを使用する場合はこちらに設定します。

MSXπ typeA用のファームウェアはこちら
https://github.com/piigaa-densetu-two-dai/MSXpi_typeA

＊SOUNDスイッチ「1」側 -> MSXπ✨モード
MSXπ✨用のファームウェアを使用する場合はこちらに設定します。

御注意：
ファームウェア書き込み時は書き込むファームウェアに関わらずSOUNDスイッチを「1」側にしておいてください。
スイッチを正しく設定してからMSXを通電して下さい。
間違えても壊れないようにしたつもりですがお気を付けください。


現時点で以下の機能に対応しています。詳細は各ディレクトリ以下のreadme.txtを読んでください。

256in1_scc：
256in1(ロム魂)のSCC音声出力対応版
CASファイルと32Kを超えるノーマルロム(主にピーガー伝説のsg1000.exeで作成したロム)には対応しません。
CASファイルや32Kを超えるノーマルロムを使用する場合はMSXπ完全互換モードでMSXπ typeA用の256in1を使用してください。

ym2413：
FM音源(MSX-MUSIC又はFMPAC)エミュレーター
FMPACのPAC機能(バックアップ機能)は機能しません

scc：
SCC音源エミュレーター
SCCゲームカセットからROMを取り除いたものに相当します。sofarun等で使用出来ます。

scc+：
スナッチャー/SDスナッチャーカセットエミュレーター
スナッチャー/SDスナッチャーの起動を確認しただけなのでゲームが正常にプレイできるか分かりません。
SCC下位互換機能は端折っています。SCC機能が必要な場合はsccファームウェアの方を使ってください。
